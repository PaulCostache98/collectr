Welcome!

Guidelines:

After registering an account, a user:
1. Starts with 10 tokens to spend on their first quest.
2. Has access to the shop, the games, the scoreboard and their account page(accessible by clicking their name in the top right corner).
3. Will be able to complete SYSTEM quests daily. These quests are pre-generated, as well as created by Admins.
4. Be free to play the games as they wish, however progress will only count if the user has bought a quest.
Once the user has gained enough money and completed quests, they will start earning badges.
Badges can be earned by completing quests, or bought with tokens. This is necessary if a user will want to create their own quests.
Creating a quest will require at least 50 tokens and 3 earned badges. However, this is the only way for users to generate passive income.
A user-generated quest, when bought, will redirect those tokens to the creator. This means that newer players will wish to buy quests from older users, especially given SYSTEM quests are reset only daily.
